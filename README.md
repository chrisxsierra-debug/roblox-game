## How to Contribute

1. **Got a game idea?** Open [GAME_DESIGN_DOC.md](docs/GAME_DESIGN_DOC.md) and fill in the sections.
2. **Planning work?** Add tasks to [ROADMAP.md](docs/ROADMAP.md) under the right phase.
3. **Joining the team?** Add yourself to [TEAM_ROLES.md](docs/TEAM_ROLES.md) with your GitHub and role.
4. Make changes directly on GitHub or clone the repo locally and push.
