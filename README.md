# Roblox Game — World Cup Collection Clicker

A World Cup–themed collection clicker for Roblox: click/AFK to earn currency, open packs to roll star football players across rarity tiers, and build a collection of your favorites.

## Status
Starting fresh. Designing the core loop and economy before building.

## Core loop (planned)
- Click / AFK to earn coins
- Spend coins on packs → RNG reveal of players by rarity (Common → GOAT)
- Collection book + player display
- Rebirth/prestige + leaderboards for retention

## Tooling
- Roblox Studio MCP server (`robloxstudio`) is configured in `.mcp.json` for live Studio access.
