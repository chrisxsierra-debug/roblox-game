# ISHIMURA

> **Co-op Sci-Fi Survival Horror on Roblox**

You and your crew respond to a distress signal on an abandoned deep-space mining vessel. The crew has been slaughtered and mutated into Necromorphs. Survive. Find the exit. **Aim for the limbs.**

## Core Features

- **Dismemberment combat** — shoot limbs, not center mass
- **2–4 player co-op** — revive teammates, share resources
- **Necromorph AI** — pathfinding enemies that change behavior when missing limbs
- **Scarce resources** — ammo and health are limited and shared
- **Atmospheric horror** — near-total darkness, spatial audio, zero HUD
